### 5. [API 문서](./05_API_문서.md)
**대상**: 백엔드 개발자, 프론트엔드 개발자, API 사용자

**내용**:
- 기본 정보 (Base URL, 인증 방식)

- 레이아웃 API (현재 레이아웃, 저장, 목록, 복원, 검증)
- WebSocket 이벤트 (클라이언트 → 서버, 서버 → 클라이언트)
- 에러 코드

**읽어야 하는 경우**:
- API를 호출하는 경우
- API 엔드포인트를 추가하는 경우
- WebSocket 이벤트를 사용하는 경우
- API 응답 형식을 확인하는 경우
