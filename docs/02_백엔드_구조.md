## 아키텍처 개요

백엔드는 **FastAPI**를 기반으로 하는 RESTful API와 **Socket.IO**를 통한 실시간 통신을 제공합니다.

```
클라이언트 요청
    ↓
FastAPI 미들웨어 (CORS, 인증)
    ↓
API 라우터 (auth, projects, layouts)
    ↓
비즈니스 로직 (core)
    ↓
데이터베이스 (SQLAlchemy ORM)
    ↓
SQLite / PostgreSQL / MySQL
```

## 핵심 모듈

### 1. main.py - FastAPI 애플리케이션

**역할**: FastAPI 앱 초기화, 미들웨어 설정, 라우터 등록

```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
import socketio

app = FastAPI(title="Furniture Platform API")

# CORS 설정
app.add_middleware(
    CORSMiddleware,
    allow_origins=settings.origins_list,
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# API 라우터 등록
app.include_router(auth.router, prefix="/api/v1/auth")
app.include_router(projects.router, prefix="/api/v1/projects")
app.include_router(layouts.router, prefix="/api/v1")

# Socket.IO 통합
socket_app = socketio.ASGIApp(websocket.sio, app)
```

**주요 기능**:
- CORS 설정으로 프론트엔드 접근 허용
- API 라우터 등록
- Socket.IO 서버 통합
- Health check 엔드포인트

### 2. config.py - 환경 설정

**역할**: 환경 변수 로드 및 설정 관리

from pydantic_settings import BaseSettings

class Settings(BaseSettings):

**주요 설정**:

- `ALLOWED_ORIGINS`: CORS 허용 오리진

### 3. database.py - 데이터베이스 연결

**역할**: SQLAlchemy 엔진 생성 및 세션 관리

```python
from sqlalchemy import create_engine
from sqlalchemy.orm import sessionmaker
from sqlalchemy.ext.declarative import declarative_base

engine = create_engine(
    settings.DATABASE_URL,
    connect_args={"check_same_thread": False}  # SQLite용
)

SessionLocal = sessionmaker(bind=engine)
Base = declarative_base()

def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```

**주요 기능**:
- DB 엔진 생성
- 세션 팩토리 생성
- `get_db()` 의존성 함수 (FastAPI에서 사용)

**확장성**:
- SQLite → PostgreSQL/MySQL 전환 시 `DATABASE_URL`만 변경
- 모든 모델은 SQLAlchemy ORM 표준 사용

### 4. core/security.py - 보안

**역할**: JWT 토큰 생성/검증, 비밀번호 해싱

```python
from passlib.context import CryptContext
from jose import jwt

pwd_context = CryptContext(schemes=["bcrypt"])

def get_password_hash(password: str) -> str:
    return pwd_context.hash(password)

def verify_password(plain: str, hashed: str) -> bool:
    return pwd_context.verify(plain, hashed)

def create_access_token(data: dict) -> str:
    payload = data.copy()
    payload.update({"exp": datetime.utcnow() + timedelta(...)})
    return jwt.encode(payload, SECRET_KEY, algorithm=ALGORITHM)
```

**주요 기능**:
- 비밀번호 해싱 (bcrypt)
- JWT 토큰 생성
- JWT 토큰 검증

### 5. core/logging.py - 중앙 집중식 로깅

**역할**: 애플리케이션 전체 로깅 설정

```python
import logging
import sys

logger = logging.getLogger("furniture_platform")

def setup_logging(level: str = "INFO") -> None:
    log_level = getattr(logging, level.upper(), logging.INFO)
    formatter = logging.Formatter(
        "%(asctime)s - %(name)s - %(levelname)s - %(message)s",
        datefmt="%Y-%m-%d %H:%M:%S"
    )
    console_handler = logging.StreamHandler(sys.stdout)
    console_handler.setLevel(log_level)
    console_handler.setFormatter(formatter)
    logger.setLevel(log_level)
    logger.handlers.clear()
    logger.addHandler(console_handler)

def get_logger(name: str = None) -> logging.Logger:
    if name:
        return logging.getLogger(f"furniture_platform.{name}")
    return logger
```

**주요 기능**:
- 타임스탬프, 로거 이름, 레벨 포함 형식화
- 모듈별 로거 생성 (`get_logger("catalog")`)
- 콘솔 출력 핸들러

## 데이터베이스 모델

### 1. User (사용자)

```python
class User(Base):
    __tablename__ = "users"
    
    id = Column(Integer, primary_key=True)
    email = Column(String, unique=True, index=True)
    password_hash = Column(String)
    full_name = Column(String, nullable=True)
    is_active = Column(Boolean, default=True)
    created_at = Column(DateTime, server_default=func.now())
    updated_at = Column(DateTime, onupdate=func.now())
    
    # Relationships
    projects = relationship("Project", back_populates="owner")
```

**필드**:
- `id`: 기본 키
- `email`: 이메일 (고유, 인덱스)
- `password_hash`: 해시된 비밀번호
- `full_name`: 사용자 이름 (선택)
- `is_active`: 활성 상태
- `created_at`, `updated_at`: 타임스탬프

---

### 2. Project (프로젝트)

```python
class Project(Base):
    __tablename__ = "projects"
    
    id = Column(Integer, primary_key=True)
    owner_id = Column(Integer, ForeignKey("users.id", ondelete="CASCADE"))
    name = Column(String)
    description = Column(String, nullable=True)
    room_width = Column(Float)
    room_height = Column(Float)
    room_depth = Column(Float)
    
    # PLY file support
    has_ply_file = Column(Boolean, default=False, nullable=False)
    ply_file_path = Column(String, nullable=True)
    ply_file_size = Column(Integer, nullable=True)
    
    created_at = Column(DateTime, server_default=func.now())
    updated_at = Column(DateTime, onupdate=func.now())
    
    # Relationships
    owner = relationship("User", back_populates="projects")
    layouts = relationship("Layout", back_populates="project")
```

**필드**:
- `id`: 기본 키
- `owner_id`: 소유자 ID (외래 키)
- `name`: 프로젝트 이름
- `description`: 설명 (선택)
- `room_width`, `room_height`, `room_depth`: 방 크기
- `has_ply_file`: PLY 파일 존재 여부
- `ply_file_path`: PLY 파일 저장 경로
- `ply_file_size`: PLY 파일 크기 (bytes)
- `created_at`, `updated_at`: 타임스탬프

### 3. Layout (레이아웃)

```python
class Layout(Base):
    __tablename__ = "layouts"
    
    id = Column(Integer, primary_key=True)
    project_id = Column(Integer, ForeignKey("projects.id", ondelete="CASCADE"))
    version = Column(Integer)
    furniture_state = Column(JSONEncodedDict)  # JSON as TEXT
    is_current = Column(Boolean, default=False)
    created_at = Column(DateTime, server_default=func.now())
    
    # Relationships
    project = relationship("Project", back_populates="layouts")
```

**필드**:
- `id`: 기본 키
- `project_id`: 프로젝트 ID (외래 키)
- `version`: 버전 번호
- `furniture_state`: 가구 상태 (JSON)
- `is_current`: 현재 레이아웃 여부
- `created_at`: 생성 시간

### 4. History (히스토리)

```python
class History(Base):
    __tablename__ = "history"
    
    id = Column(Integer, primary_key=True)
    layout_id = Column(Integer, ForeignKey("layouts.id", ondelete="CASCADE"))
    user_id = Column(Integer, ForeignKey("users.id", ondelete="CASCADE"))
    change_type = Column(String)  # 'add', 'move', 'delete', 'update'
    before_state = Column(JSONEncodedDict, nullable=True)
    after_state = Column(JSONEncodedDict, nullable=True)
    timestamp = Column(DateTime, server_default=func.now())
```

**필드**:
- `id`: 기본 키
- `layout_id`: 레이아웃 ID (외래 키)
- `user_id`: 사용자 ID (외래 키)
- `change_type`: 변경 유형
- `before_state`, `after_state`: 변경 전후 상태
- `timestamp`: 변경 시간
