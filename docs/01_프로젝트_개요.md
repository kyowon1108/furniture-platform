## 프로젝트 소개

**3D 가구 배치 플랫폼**은 실시간 협업이 가능한 웹 기반 3D 가구 배치 도구입니다. 사용자는 직관적인 인터페이스를 통해 방의 크기를 설정하고, 다양한 가구를 배치하며, 실시간으로 다른 사용자와 협업할 수 있습니다.

### 핵심 가치
- **실시간 협업**: WebSocket을 통한 즉각적인 변경사항 동기화
- **직관적인 UI**: 드래그 앤 드롭, 키보드 단축키 지원
- **자동 저장**: 5초마다 자동 저장으로 작업 손실 방지
- **충돌 감지**: 가구 간 충돌 자동 감지 및 시각화
- **다양한 도구**: 측정, 조명 시뮬레이션, Undo/Redo 등
- **3D 스캔 지원**: PLY 파일 업로드로 실제 방 모델 사용 가능

## 기술 스택

### 백엔드
```
언어: Python 3.9.18
프레임워크: FastAPI
데이터베이스: SQLite (개발), PostgreSQL/MySQL (확장 가능)
ORM: SQLAlchemy
인증: JWT + bcrypt
실시간 통신: Socket.IO
테스트: pytest
```

### 프론트엔드
```
프레임워크: Next.js 14 (App Router)
UI 라이브러리: React 18
언어: TypeScript
3D 엔진: Three.js + React Three Fiber
상태 관리: Zustand
스타일링: Tailwind CSS
실시간 통신: Socket.IO Client
```

### 개발 도구
```
패키지 관리: Conda (백엔드), npm (프론트엔드)
버전 관리: Git
API 문서: Swagger UI (자동 생성)
```

## 개발 마일스톤

### Milestone 1: 기본 기능 (완료)
**목표**: 핵심 기능 구현
- 사용자 인증 시스템
- 프로젝트 및 레이아웃 관리
- 3D 씬 렌더링
- 실시간 협업
- 충돌 감지

**기간**: 초기 개발
**결과**: 30개 백엔드 파일, 25개 프론트엔드 파일, ~4,500줄 코드

### Milestone 2: 필수 기능 (완료)
**목표**: 사용자 경험 향상
- 자동 저장 시스템
- 가구 카탈로그
- 드래그 앤 드롭
- PNG 내보내기
- 프로젝트 대시보드

**기간**: Milestone 1 이후
**결과**: 7개 신규 파일, 8개 업데이트 파일, ~1,500줄 추가 코드

### Milestone 3: 고급 기능 (완료)
**목표**: 전문가 도구 추가
- Undo/Redo 시스템
- 측정 도구
- 조명 시뮬레이션
- 복사/붙여넣기
- 키보드 단축키 확장
- PLY 파일 업로드 (3D 스캔 모델 지원)

**기간**: Milestone 2 이후
**결과**: 10개 신규 파일, 10개 업데이트 파일, ~2,500줄 추가 코드
