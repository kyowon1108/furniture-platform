# 프로젝트 개요

## 문제 정의

- 방 구조 설계와 가구 배치를 한 번에 다루는 가벼운 웹 편집 흐름이 필요했습니다.
- 팀 단위로 같은 공간을 같이 보고 수정하는 협업 기능도 같이 필요했습니다.
- 초기 진입 비용을 낮추기 위해 설치형 툴보다 브라우저 기반 흐름을 우선했습니다.

## 대상 사용자

- 원룸/작은 방 구조를 빠르게 시뮬레이션하려는 사용자
- 간단한 인테리어 배치 결과를 바로 보고 싶은 사용자
- 같은 공간을 동시에 검토해야 하는 협업 사용자

## 핵심 가치

- 브라우저에서 바로 시작 가능한 방 구조 설계
- 3D 가구 배치와 충돌 감지
- 실시간 협업, presence, 잠금
- 업로드 파일에 대한 보호된 다운로드 경로

## 현재 제품 범위

- 프로젝트 생성 / 조회 / 수정 / 삭제
- Room Builder 기반 방 구조 설계
- Free Build 기반 타일형 구조 설계
- GLB / PLY 업로드
- 3D 에디터에서 가구 배치, 이동, 회전, 삭제
- 레이아웃 저장 / 버전 관리
- 관리자 allowlist 기반 카탈로그 관리
