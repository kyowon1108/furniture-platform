# 카메라 타겟 위치 개선

## 🎯 문제
줌인 시 카메라가 바닥(Y=0)을 중심으로 회전하여 가구 배치가 불편함

## ✅ 해결
OrbitControls의 타겟을 바닥에서 1m 위로 설정

```typescript
<OrbitControls
  target={[0, 1, 0]}  // 가구 중심 높이
  minDistance={0.5}
  maxDistance={50}
/>
```

## 💡 효과

### 수정 전 (타겟: 바닥)
```
줌인 → 바닥을 보게 됨 → 가구가 화면 위쪽에 위치 → 배치 불편 ❌
```

### 수정 후 (타겟: 1m 높이)
```
줌인 → 가구를 보게 됨 → 가구가 화면 중앙에 위치 → 배치 편리 ✅
```

## 🎨 장점

1. **자연스러운 시점**
   - 가구 높이에 맞춰진 시선
   - 사람의 앉은 눈높이와 유사
   - 직관적인 작업 환경

2. **편리한 가구 배치**
   - 가구가 항상 화면 중앙
   - 정확한 위치 조정 가능
   - 다른 가구와의 관계 파악 용이

3. **효율적인 작업**
   - 불필요한 카메라 조정 감소
   - 빠른 배치 작업
   - 피로도 감소

## 📊 타겟 높이별 용도

| 높이 | 용도 | 적합한 작업 |
|------|------|------------|
| 0m (바닥) | 바닥 재질 디자인 | 타일, 카펫 선택 |
| **1m (가구)** | **일반 작업** | **가구 배치 ✅** |
| 1.5m (눈높이) | 프레젠테이션 | 고객 시연 |
| 2.5m (천장) | 천장 디자인 | 조명 배치 |
